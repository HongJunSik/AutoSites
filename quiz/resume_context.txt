[더웰니스의원 퀴즈 프로그램 - 작업 컨텍스트 요약]

본 퀴즈 프로그램은 더웰니스의원 시그니처 시술(리팟, 쥬브젠, 울쎄라피, 써마지, 덴서티, 보톡스)을 위한 Duolingo 스타일의 학습용 웹 앱입니다.

1. 기술 스택 및 구조
- 프론트엔드: Vanilla HTML / Vanilla CSS / Vanilla JS (Tailwind 없음)
- 데이터베이스: Firebase Firestore compat SDK (v10.8.0) 연동 + 로컬 스토리지 Fallback 지원
- 로컬 서버: http-server (포트 8080) 구동 중

2. 핵심 기능 및 구현 세부사항
- 야돈(Slowpoke) 마스코트: 유저 XP 성장에 따라 100 XP 단위로 악세서리 및 몸집이 발달하는 SVG 그래픽 오버레이 시스템 구축 (최종 1000 XP). 300 XP 이상 시 덤벨 리프팅(Hobby), 식사(Eating), 수면(Sleeping), 혼잣말 등 백그라운드 애니메이션 및 명대사 자동 순환(Idle Loop) 탑재.
- 사용자 인증: 닉네임과 패스워드를 매칭하여 Firestore 'users' 컬렉션에 유저 데이터를 유지하며 로그인 진행.
- 새로고침 세션 유지: 페이지 새로고침 시 Firebase 로딩 레이턴시로 인한 강제 로그아웃 문제를 방지하기 위해, sessionStorage를 이용해 로컬 데이터를 동기식으로 먼저 로드한 뒤 백그라운드에서 Firestore 비동기 동기화를 적용하는 2단계 인증 아키텍처 도입.
- 문제 은행 (10문항 시스템): 각 시술 카테고리당 문제를 3개에서 10개(총 60개)로 전면 확장 및 전문 해설 작성.
- 동적 보기 셔플: 객관식(multiple-choice) 및 빈칸 채우기(fill-blank) 시 정답이 항상 1번에 위치하는 쏠림 현상을 차단하기 위해, 렌더링 시점에 선택지 배열을 무작위 셔플링하도록 개선.
- 경험치 시스템 수정: 퀴즈 완주 시 정답 개수와 관계없이 문제당 3 XP씩 총 30 XP를 지급하며, 이미 100% 완료된 진도를 재학습 시 경험치를 획득할 수 없도록 '+0 XP' 잠금 적용.
- 자동 데이터 시딩: 로컬의 10문항 스키마 확장을 감지하면 Firebase의 이전 3문항 데이터를 자동으로 오버라이팅하여 강제 업그레이드(Force-seed)하는 스마트 DB 싱크 기능 개발.

3. 향후 이어서 진행할 작업 가이드
"이 resume_context.txt를 복사해서 새로운 대화 세션의 첫 프롬프트로 전달하면 불필요한 토큰 소비 없이 이전 컨텍스트를 완벽히 로드할 수 있습니다."
